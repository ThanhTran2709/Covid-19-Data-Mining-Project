## Libraries and Packages Used:

**scikit-learn:** A Python machine learning library used to build and evaluate models.\
**Pandas:** Used for data manipulation and analysis.\
**NumPy:** Used for array operations.\
**Matplotlib:** Creating data visualizations.\
**Seaborn:** More data visualizations.\
**Geopandas:** Manipulation of geospatial data.\
**Datetime:** Classes used for manipulation of dates and times.\
**scikit-optimize:** Used for BayesSearchCV hyperparameter tuning.\
**thefuzz:** Fuzzy string matching, used in preparing data.\
**scipy:** Used specifically for the scipy.stats.entropy function, which calculates the entropy of a distribution.\
**joblib:** For parallel computing on large datasets.\
**csv:** Used to write output to a .csv file.
